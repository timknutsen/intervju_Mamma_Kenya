# Archive

This folder holds material that is preserved for reference but is not part of the active workflow.

Why it exists:

- to keep the active repo structure focused on the current English-first process
- to retain recoverable history without making old experiments look current
- to make it easy to start a new session from the real active baseline

What belongs here:

- retired workflow artifacts
- obsolete editorial compatibility files
- old experiments that may still be useful for reference

Subfolders:

- `legacy-editorial/`: retired story and transcript-support files from older workflow stages
- `mau-mau-pilot/`: the original Mau Mau test experiment

Archived files should not be treated as current source of truth unless the user explicitly brings them back into active use.
